Algorithm

  1. dist[source] = 0; all others = ∞
  2. push source into priority queue
  3. while queue not empty:
  4. u = pop cell with min cost
  5. if u == target: done!
  6. mark u visited
  7. for each neighbor v of u:
  8. if v is wall: skip
  9. alt = dist[u] + terrain_cost(v)
  10. if alt < dist[v]:
  11. dist[v] = alt; prev[v] = u
  12. push / update v in queue
  13. reconstruct path from prev[]
