Algorithm

  1. dist[source] = 0; all others = ∞
  2. push source into priority queue
  3. while queue not empty:
  4. u = pop node with min dist
  5. mark u visited
  6. for each neighbor v of u:
  7. alt = dist[u] + weight(u, v)
  8. if alt < dist[v]:
  9. dist[v] = alt; prev[v] = u
  10. push / update v in queue
  11. reconstruct path from prev[]
